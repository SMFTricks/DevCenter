[size=4][color=#bf307a]Introduction[/color][/size]
[size=1em]Dev Center adds a couple of settings to the admin panel that might be useful when developing a modification.[/size]

[size=3][color=#1f4882]Features[/color][/size]
[list]
[li][size=0.9em]Enable and display debugging information.[/size][/li]
[li][size=0.9em]Adds another phpinfo() page in a phpinfo action for easy and quick access.[/size][/li]
[li][size=0.9em]Monitor server load in the footer.[/size][/li]
[/list]
